## 🌐 Deployment

The website can be deployed using **GitHub Pages**:

1. Fork or clone this repository.
2. Push all files to the `main` branch.
3. Enable **GitHub Pages** from repository settings.
4. Ensure the **root folder** is selected for publishing.
5. The site will be available at `https://<username>.github.io/<repository>/`.

## ⚡ Features

- Premium SaaS-style design with modern hero section, gradient buttons, and hover effects.
- Fully responsive and mobile-first.
- Scroll fade-in animations for polished, professional look.
- SEO-ready with meta tags, `robots.txt`, and `sitemap.xml`.
- Multi-project portfolio layout for:
  - **Axyz Order System**
  - **Conzeus Place**
  - **My-Edoc Generator**
- Contact form (mailto-based) for quick inquiries.
