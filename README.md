## 📌 Overview

Axyz Display System is a lightweight, zero-hardware restaurant order display solution designed to replace shouting order numbers and reduce crowding at pickup counters.

The system runs entirely from a website and can be deployed globally using **GitHub Pages + Cloudflare** with **zero hosting cost**.

Restaurants only need:

- a phone, a tablet, and a TV or browser screen
- internet connection
- modern web browser

No installation required.

---

## 🎯 Purpose

The system provides a real-time visual workflow:

1. Cashier registers paid orders
2. Kitchen prepares food
3. Waiter marks orders as READY
4. Customers see pickup numbers on display
